# Phase 1: Git & GitHub

## Key Learnings
* **Git vs GitHub:** Git is local; GitHub is the cloud host.
* **SSH Keys:** Used for secure authentication without passwords.
* **The Workflow:** `add` -> `commit` -> `push`.

## Commands Used
- `git clone`: Copy repo to local.
- `git add .`: Stage changes.
- `git commit -m ""`: Snapshot changes.
- `git push`: Upload to cloud.
